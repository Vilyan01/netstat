# Netstat

A Ruby interface for getting data from /proc/net/tcp like netstat.

## Acknowledgments

Inspired by https://github.com/davidw/ruby-netstat.

## Installation

Add this line to your application's Gemfile:

    gem 'netstat'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install netstat

## Usage

    Netstat.read_tcp
